Password reset

Here's another project of mine, which involves handling password change.

It is important to create this kind of mechanic in a website because it is used throughout the industry.

The mechanics are very simple, the user must enter their email which is associated with their account.

The latter will receive an email with a temporary link. Clicking on the link sends the user to a new page to make the change.

At the end, the user is prompted to return to the login page or return to the site's home page.
